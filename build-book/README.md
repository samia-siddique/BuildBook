# BuildBook

BuildBook is a full stack productivity tracker that I built to organize my coding journey.

It allows users to:
. Create projects
. Track coding sessions
. Set weekly goals
. View progress from a dashboard which is in progress right now

## Why I built it

I wanted a place where I could keep track of everything I learn while studying programming. Instead of writing notes in different places, BuildBook keeps my projects, sessions, and goals together.

## Tech Stack

. React
. Node.js
. Express
. MongoDB
. React Router
. CSS
. Lucide Icons

## Features

. Dashboard
. Project Tracker
. Goal Tracker
. Session Tracker
. Responsive UI
